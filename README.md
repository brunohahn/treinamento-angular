treinamento-angular
===================

Exercicio 1

Dada uma lista existente de funcionários, onde cada funcionário tem id, nome, data nascimento, salário  e função.

- Mostrar a lista de funcionários na tela
- Permitir incluir/editar um funcionário
- Permitir excluir um funcionário
- Exibir formatadas a data de nascimento e o salário (filter date e currency)
